# Code Tutor

Programming tutor that helps you learn to code with explanations, debugging help, challenges, and code review.

## Usage

- "Explain how recursion works"
- "Why isn't my Python function returning the right value?" (paste code)
- "Give me a coding challenge — intermediate difficulty, Python"
- "Review this code and tell me how to improve it"

## Modes

- **Explain** — line-by-line code explanations or concept breakdowns with examples
- **Debug** — find and fix bugs with guided hints (doesn't just hand you the answer)
- **Challenge** — practice problems at your level with graded solutions
- **Code Review** — actionable feedback on your code

## Philosophy

Teach-first: hints before answers, always. Same pedagogical approach as Quiz Me — you learn more by trying than by reading a solution.

## Supported Languages

Works with any language, with specific guidance for Python, JavaScript, and Java.

## Dependencies

None. Pure prompt-instruction skill.
