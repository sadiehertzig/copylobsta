# Academic Deep Research

Transparent, rigorous research with full methodology — not a black-box API wrapper. Conducts exhaustive investigation through mandated 2-cycle research per theme, APA 7th citations, evidence hierarchy, and 3 user checkpoints.

> **Original author:** [kesslerio](https://github.com/kesslerio/academic-deep-research-clawhub-skill)
> Installed from [ClawHub](https://clawhub.ai) — `academic-deep-research` v1.0.0

## Usage

```
/research "Comprehensive analysis of [your topic]"
```

Or ask for "deep research on..." or "exhaustive analysis of..."

## How It Works

### Phase 1: Clarification
Agent asks 2-3 essential questions and confirms understanding before starting.

### Phase 2: Planning
Agent presents major themes (3-5) and an execution plan. **You approve before execution begins.**

### Phase 3: Execution (Auto)
For each theme, two full research cycles:
- `web_search` (count=20) for broad landscape
- Analysis, gap identification, and source fetching
- Synthesis, assumption challenging, and deeper dive

### Phase 4: Report
Academic narrative with Executive Summary, Knowledge Development, Comprehensive Analysis, Practical Implications, and APA References.

## Key Features

- **Full methodology visible** — every step documented and reproducible
- **No external dependencies** — runs entirely on OpenClaw native tools
- **3 user checkpoints** — you approve scope, plan, and final report
- **Academic rigor** — APA 7th edition citations, evidence hierarchy, confidence levels
- **Mandated 2 research cycles per theme** — no single-pass shortcuts

## Evidence Hierarchy

1. Systematic reviews and meta-analyses (highest)
2. Randomized controlled trials
3. Cohort/longitudinal studies
4. Expert consensus/guidelines
5. Cross-sectional/observational
6. Expert opinion/editorials
7. Media reports/blogs (lowest)

## When to Use This

- Literature reviews requiring academic rigor
- Competitive intelligence with source verification
- Complex topics needing multi-source synthesis
- Any research where you need to show your work
