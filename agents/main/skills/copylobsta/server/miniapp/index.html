Anthropic API Key

Anthropic makes Claude — one of the best AI models out there. It'll power your bot's smartest conversations.

Step 1: Set up API access

Even if you use claude.ai, you need a separate API account to power your bot.

  1. Open the Anthropic Console and sign up or sign in Open Anthropic Console
  2. Go to Settings → Billing and add a payment method

New accounts get $5 in free credits. Adding a payment method keeps your bot running after those run out.

Estimated monthly spend

  • Light use (a few chats/day): ~$5-10/month
  • Regular use (daily homework help, quizzes): ~$10-20/month
  • Heavy use (long sessions, creative writing): ~$20-40/month

Step 2: Set a spend limit

Cap your spending so there are no surprises.

  1. Open your usage limits Open Usage Limits
  2. Set your monthly spend limit — we recommend starting at $25/month

You can always change this later.

Step 3: Create your API key

  1. Open the API Keys page Open API Keys
  2. Click Create Key
  3. Name it anything (e.g., "my-bot")
  4. Copy the key — you won't see it again!

Google Gemini API Key

Gemini is Google's AI — fast, affordable, and great at following instructions. It offers models from Nano (lightweight) to Flash (balanced) to Pro (most capable). Your bot uses it alongside Claude and GPT for different tasks.

Step 1: Set up API access

You probably have a Google account already — but you still need to enable API access through Google AI Studio.

  1. Open Google AI Studio and sign in with your Google account Open Google AI Studio
  2. Accept the API terms of service
  3. Optionally: set up billing for higher rate limits (Settings → Billing)

Gemini has a free tier that works for personal use. Paid tier gives faster responses and higher limits.

Estimated monthly spend

  • Free tier: $0/month — 15 requests/minute, works for casual use
  • Light paid use: ~$2-5/month
  • Regular use: ~$5-15/month
  • Heavy use: ~$15-30/month

Step 2: Set a spend limit

If you enabled billing, set a budget. If you're on the free tier, skip this.

  1. Open Google Cloud billing budgets Open Billing Budgets
  2. Create a budget — we recommend starting at $15/month

Step 3: Create your API key

  1. Open the API Keys page Open API Keys
  2. Click Create API Key
  3. Select a Google Cloud project (or let it create one)
  4. Copy the key

Set a monthly budget so your credits last.

  1. Open your billing settings Open Billing Settings
  2. Set a monthly budget limit — we recommend $20/month

Step 3: Create your API key

Advanced: Set up SSH access

Your bot is managed via AWS SSM (no SSH required). But if you want to SSH in directly — for example, to use VS Code Remote SSH or SCP files — you can set up a key pair.

  1. Open the EC2 Key Pairs page in AWS Console Open Key Pairs
  2. Click Create key pair
  3. Name it something memorable (e.g., my-bot-key)
  4. Choose .pem format and click Create
  5. Your browser will download a .pem file — move it somewhere safe

Rename & secure your key file

Move the file to your SSH folder and lock down permissions so only you can read it:

mv ~/Downloads/my-bot-key.pem ~/.ssh/my-bot-key.pem
chmod 600 ~/.ssh/my-bot-key.pem

On Windows, right-click the .pem file → Properties → Security → remove all users except yourself.

Attach the key to your instance

  1. Open your EC2 instances Open Instances
  2. Select your instance, then Actions → Instance Settings → Attach/Replace Key Pair
  3. Choose the key pair you just created and attach it

Open port 22 in the security group

  1. In the instance details, click the Security tab
  2. Click on the security group link
  3. Click Edit inbound rules
  4. Add a rule: Type = SSH, Source = My IP
  5. Click Save rules

Using "My IP" restricts access to your current network. You'll need to update this if your IP changes.

Connect!

Now you can SSH in:

ssh -i ~/.ssh/my-bot-key.pem ubuntu@YOUR_INSTANCE_IP

Replace the IP with your instance's public IP from the AWS Console.
